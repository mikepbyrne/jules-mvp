Jules SMS Opt-In Process

Last Updated: December 29, 2025

How Users Consent to Receive Messages

Jules collects explicit consent from users before sending any SMS messages. We offer two opt-in methods:

Method 1: Web Registration (Primary)

Users opt-in to SMS messaging during account creation on our web application:

Registration Process:

  1. User visits the Jules web application
  2. User creates an account and provides their phone number
  3. User explicitly checks a consent checkbox with the following text:

Message and data rates may apply. You can opt-out at any time by replying STOP. Reply HELP for assistance.

  1. Upon successful registration, user receives a confirmation SMS

Method 2: SMS Keyword Opt-In (Alternative)

Users can also opt-in by texting specific keywords to our toll-free number:

Supported Keywords:

  • START
  • HI
  • HELLO
  • JOIN
  • SUBSCRIBE
  • YES

Process:

  1. User texts any keyword above to: (866) 497-8083
  2. Jules responds with opt-in confirmation message
  3. User's consent is logged with timestamp

Opt-In Confirmation Message

After opting in through either method, users receive this confirmation:

"Welcome to Jules! You're now opted-in to receive meal planning messages. Reply with recipe ideas or HELP for options. To opt-out anytime, reply STOP. Msg&data rates may apply."

What Users Are Told at Opt-In

  • Purpose: Meal planning assistance, recipe management, shopping lists
  • Frequency: 10-50 messages per week (varies by user activity)
  • Cost: Standard message and data rates may apply
  • Opt-Out: Reply STOP at any time to unsubscribe
  • Help: Reply HELP for assistance

Message Types Users Will Receive

Conversational Messages:

  • Onboarding conversations (collecting household info, dietary restrictions)
  • Recipe management (adding, viewing, searching recipes)
  • Meal planning assistance (weekly planning, suggestions)
  • Shopping list generation
  • Responses to user questions

Characteristics:

  • All messages are transactional or conversational
  • Messages only sent in response to user activity
  • No promotional or marketing content
  • AI-powered natural language responses

Opt-Out Process

Users can opt-out at any time by:

  • Replying STOP to any message from Jules
  • Replying UNSUBSCRIBE
  • Replying CANCEL
  • Replying END
  • Replying QUIT

Opt-Out Confirmation Message:

"Got it! You're all set - no more messages from Jules. Your data will be deleted within 30 days. If you change your mind, just text START. Take care!"

Help Message

Users can get help at any time by replying HELP:

"Jules helps with meal planning! Commands: RECIPES (view recipes), SUGGEST (get meal ideas), PROFILE (household info), STOP (unsubscribe). Or just chat naturally with me!"

Data Collection and Privacy

  • Phone numbers are stored securely and never shared with third parties
  • All opt-in/opt-out actions are logged with timestamps
  • User data is deleted within 30 days of opt-out
  • Messages comply with TCPA and CTIA guidelines

Consent Record Keeping

For each user opt-in, we log:

  • Phone number (E.164 format)
  • Opt-in timestamp (ISO 8601 format)
  • Opt-in method (web_registration or sms_keyword)
  • Opt-in source (keyword used if SMS opt-in)
  • IP address (for web registration)
  • Current opt-in status (active, paused, opted_out)
